## Como funciona

O usuário acessa um website, preenche um formulário com seus dados e o sistema gera automaticamente uma proposta personalizada.

**Fluxo:**
```
Usuário preenche formulário → Dados são processados → Proposta é gerada com os dados do usuário
```
